# NobetArasi# Nöbet Arası

Nöbet Arası, intern döneminde ve doktorluk pratiğinde kısa molalarda klinik bilgi tekrarı yapılabilmesi için geliştirilen mobil quiz uygulamasıdır.

## İlk MVP Kapsamı

- Kullanıcı kayıt olur.
- Kullanıcı giriş yapar.
- 4 kategoriden birini seçer.
- 10 soruluk quiz çözer.
- Doğru cevaplara göre XP kazanır.
- XP'ye göre level ilerlemesi görür.

## İlk Kategoriler

1. Mikrobiyoloji
2. Farmakoloji
3. Patoloji
4. Dahiliye

## Teknoloji Planı

### Mobile

- React Native
- Expo
- TypeScript
- Expo Router
- TanStack Query
- React Hook Form
- Zod

### Backend

- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- JWT Authentication
